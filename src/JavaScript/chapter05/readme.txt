第五章 JavaScript常用函数
第一节：JavaScript全局函数
全局函数不属于任何一个内置对象；
JS包含以下7种全局函数，用于一些常用的功能：escape()，eval()，isNan()，isFinite()，parseFloat()，parseInt()，unescape()；
Eval()方法：执行代码

Alert();弹出消息窗   confirm()   prompt()

setTimeout();执行一次
setInterval();执行多次
open();打开窗口

Window.onload   发生在文档全部下载完毕的时候
Window.onresize 发生在窗口大小发生变化的时候
